# 粒子轨道事件系统 (ParticleTrack) 使用指南

## 🎆 概述

粒子轨道事件系统是专为CombatEditor设计的全新粒子控制系统，提供了完全跟随轨道进度的精确粒子效果控制。与传统的点事件不同，粒子轨道事件能够在整个轨道时间范围内精确控制粒子的播放、强度和位置。

### ✨ 主要特性

- **🎬 完全轨道同步**: 粒子效果完全跟随时间轴轨道进度
- **🎯 精确时间控制**: 支持进入/退出轨道时的精确控制
- **📍 灵活位置绑定**: 支持多种节点绑定和自定义位置
- **💫 动态强度控制**: 通过曲线控制粒子发射强度
- **👁️ 实时编辑器预览**: 在编辑器模式下完全可见的实时预览
- **⚙️ 高度可配置**: 丰富的配置选项满足各种需求

## 🚀 快速开始

### 1. 创建粒子轨道事件

1. 在CombatEditor中右键点击轨道区域
2. 选择 `AbilityEvents / ParticleTrack`
3. 新的粒子轨道事件将以长轨道形式出现

### 2. 基础配置

#### 粒子对象设置
- **粒子预制体**: 选择包含ParticleSystem组件的预制体
- 系统会自动检测预制体中的所有粒子系统

#### 轨道播放控制
- **进入轨道时播放**: 当时间轴进入轨道范围时自动播放
- **退出轨道时停止**: 当时间轴退出轨道范围时自动停止
- **退出时清除粒子**: 立即清除所有已发射的粒子
- **销毁延迟**: 轨道结束后延迟销毁粒子对象的时间

### 3. 调整轨道时间范围

在CombatEditor的时间轴上拖拽轨道的开始和结束点来调整粒子效果的持续时间。

## 🎛️ 高级配置

### 轨道时间控制

#### 播放速度倍率
```
1.0 = 正常速度
0.5 = 半速播放（慢动作效果）
2.0 = 双倍速度（快速效果）
```

#### 进度映射曲线
使用AnimationCurve来创建非线性的播放效果：
- **线性**: `AnimationCurve.Linear(0,0,1,1)` - 匀速播放
- **加速**: `AnimationCurve.EaseIn(0,0,1,1)` - 先慢后快
- **减速**: `AnimationCurve.EaseOut(0,0,1,1)` - 先快后慢
- **爆发式**: 自定义尖峰曲线创建爆发效果

### 位置绑定设置

#### 节点绑定
```csharp
BodyCenter    // 身体中心 - 适合全身效果
Head          // 头部 - 适合头部特效
Hand          // 手部 - 适合手部技能
Weapon        // 武器 - 适合武器特效
WeaponTip     // 武器尖端 - 适合剑尖火花等
```

#### 高级绑定选项
- **自定义目标**: 绑定到场景中指定名称的对象
- **自定义父级**: 为粒子对象指定特定的父级变换

### 粒子强度控制

启用强度控制可以通过曲线动态调整粒子发射率：

```
强度曲线值 × 最大强度倍率 × 原始发射率 = 最终发射率
```

#### 常用强度模式
- **渐强**: 从0开始逐渐增强到1
- **渐弱**: 从1开始逐渐减弱到0
- **爆发**: 短时间内达到高峰然后快速衰减
- **脉冲**: 周期性的强度变化

## 🎨 使用案例

### 案例1: 剑气特效
```
粒子预制体: SwordAura.prefab
绑定节点: Weapon
位置偏移: (0, 0, 0.5) // 剑尖前方
跟随移动: true
跟随旋转: true
播放速度: 1.0
轨道时间: 攻击动作的挥砍阶段
```

### 案例2: 充能光环
```
粒子预制体: ChargingAura.prefab
绑定节点: BodyCenter
强度控制: 启用
强度曲线: 渐强曲线 (0→1)
最大强度倍率: 3.0
轨道时间: 技能充能阶段
```

### 案例3: 冲击波效果
```
粒子预制体: ImpactWave.prefab
绑定节点: BottomCenter
跟随移动: false // 固定在冲击位置
进度曲线: 爆发式曲线
播放速度: 2.0
轨道时间: 攻击命中瞬间的短时间
```

### 案例4: 持续燃烧
```
粒子预制体: Fire.prefab
绑定节点: BodyCenter
循环播放: true
强度控制: 启用
强度曲线: 先强后弱
轨道时间: 燃烧Buff持续时间
清除粒子: false // 保持余烬效果
```

## 🔧 编辑器使用技巧

### 预览功能
- **实时预览**: 拖拽时间轴时粒子效果实时更新
- **精确模拟**: 粒子播放完全同步轨道进度
- **缩放兼容**: 支持时间轴缩放的精确预览

### 调试功能
- **启用调试日志**: 在控制台查看详细的状态信息
- **配置诊断**: 一键检查配置是否正确
- **预览控制**: 手动清除和刷新预览

### 性能优化
- **LOD系统**: 根据距离调整粒子复杂度
- **合理延迟**: 设置适当的销毁延迟时间
- **批量管理**: 避免同时播放过多粒子轨道

## ⚠️ 注意事项

### 性能考虑
1. **粒子数量**: 避免同时播放过多复杂粒子效果
2. **更新频率**: 轨道事件每帧都会更新，注意性能影响
3. **内存管理**: 合理设置销毁延迟避免内存累积

### 兼容性
- ✅ 支持所有Unity标准粒子系统特性
- ✅ 兼容现有CombatEditor轨道系统
- ✅ 支持多粒子系统复合效果
- ⚠️ 需要Unity 2019.4或更高版本

### 最佳实践
1. **预制体设计**: 将相关粒子系统组合在一个预制体中
2. **曲线设计**: 使用平滑的曲线避免突变
3. **位置偏移**: 合理设置偏移避免穿插问题
4. **测试预览**: 充分利用编辑器预览功能进行调试

## 🐛 常见问题

### Q: 粒子不显示怎么办？
A: 检查以下项目：
- 粒子预制体是否正确指定
- 预制体是否包含ParticleSystem组件
- 轨道时间范围是否正确设置
- 是否在轨道范围内播放

### Q: 粒子位置不对怎么办？
A: 确认以下设置：
- 绑定节点是否正确
- 位置偏移是否合适
- 是否启用了跟随移动
- 自定义目标对象是否存在

### Q: 预览效果与运行时不同？
A: 这可能是因为：
- 时间缩放事件的影响
- 动画速度控制的影响
- 运行时环境差异

### Q: 性能问题如何解决？
A: 尝试以下优化：
- 减少同时播放的粒子轨道数量
- 降低粒子系统的复杂度
- 使用LOD系统
- 合理设置销毁延迟
